# 단축키 가이드

## 탭 관리

| 단축키 | 기능 |
|---|---|
| `Cmd+T` | 새 탭 생성 |
| `Cmd+Shift+]` | 다음 탭으로 이동 |
| `Cmd+Shift+[` | 이전 탭으로 이동 |

## 패널 분할

| 단축키 | 기능 |
|---|---|
| `Cmd+D` | 수평 분할 (좌/우) |
| `Cmd+Shift+D` | 수직 분할 (상/하) |
| `Cmd+W` | 현재 패널 닫기 (패널 1개일 때 탭 닫기) |

## 패널 이동 - 번호

| 단축키 | 기능 |
|---|---|
| `Cmd+1` | 패널 1로 이동 |
| `Cmd+2` | 패널 2로 이동 |
| `Cmd+3` | 패널 3로 이동 |
| `Cmd+4` | 패널 4로 이동 |
| `Cmd+5` | 패널 5로 이동 |
| `Cmd+6` | 패널 6로 이동 |
| `Cmd+7` | 패널 7로 이동 |
| `Cmd+8` | 패널 8로 이동 |

## 패널 이동 - 방향

| 단축키 | 기능 |
|---|---|
| `Cmd+Option+←` | 왼쪽 패널로 이동 |
| `Cmd+Option+→` | 오른쪽 패널로 이동 |
| `Cmd+Option+↑` | 위쪽 패널로 이동 |
| `Cmd+Option+↓` | 아래쪽 패널로 이동 |

## 패널 이동 - 순차

| 단축키 | 기능 |
|---|---|
| `Ctrl+Tab` | 다음 패널 |
| `Ctrl+Shift+Tab` | 이전 패널 |
